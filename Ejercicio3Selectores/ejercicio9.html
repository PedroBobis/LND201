ejercicio 9

Cuál es mi color?



Explicación

body { font-family: Arial; color: black; }: le esta indicando que la fuente dentro del body que no este especificada en otra etiqueta sera arial y de color negro. h1 { font-family: Verdana; } indica que la fuente sera verdana y saldra de color negro por que ereda el color del body al encontrarse dentro de el y no especificarse otro color. p { color: red; } indica que el color de la fuente dentro de los p es rojo y en este caso eredara el tipo de fuente arial del body ya que no se a especificado ninguno diferente y el p se encuentra dentro del body.
